# 給料計算ツール (Excel VBA)

このExcelファイルは、時給と勤務時間からその日の給料を自動計算するVBAツールです。

※ このツールは開発タブを操作する必要はありません。
   Excelを開いて「計算」「削除」「日付」ボタンを押すだけで使用できます。

## 使い方
1. 「給料計算.xlsm」ファイルを開く
2. 上部に表示されるセキュリティ警告の「コンテンツの有効化」を押す
3. 「このファイルを信頼済みドキュメントにしますか？」→「はい」を選択
4. カレンダーの日付の下のセルをクリックすると、右側にプルダウンが表示されます
5. 勤務内容を選択します
   - 「定時」：1日8時間勤務
   - 「残業◯」：残業時間に応じて自動計算
   - 「1時間〜7時間」：遅刻・早退に対応
   - 「有給」「欠勤」「休日出勤」にも対応
6. 選択が終わったら「計算」ボタンを押すと、その日の給料が自動計算されます

## その他の機能
- 「削除」ボタン：曜日・日付をすべて初期化します
- 「日付」ボタン：その月の日付を自動入力します  
   ※ K29セルに、その月の「曜日番号」を入力しておく必要があります  
     （例：日曜日=3、月曜日=4…）  
   ※ 曜日番号は、日付を正しく並べるために使用しています。  
     不要な曜日番号が表示された場合は、選択してDeleteしてください。

## 機能概要
お試し1
勤務表のプルダウンで選択した勤務区分（定時、残業、有給、休日出勤など）に応じて、
日給を自動計算するVBAコードです。

指定範囲をループし、勤務内容ごとに計算式を切り替えて
下のセルに日給を出力します。

月間の給与合計や交通費の自動集計も行い、
ボタン1つで1ヶ月分の給与計算が完了するように作成しました。

削除
「削除」シートに用意してある初期状態のテンプレートを、
勤務表シート（4月）の C30〜I67 にコピーすることで、
日付・曜日・勤務区分・計算結果をすべてリセットする処理です。

合計
勤務表の金額が入っているセルを順番に読み取り、
すべての金額を足して M25 に合計を出力します。

日付1
このコードは、K29 に入力された曜日番号（3〜9）をもとに、
その月の1日がどの曜日から始まるかを決めて、
カレンダー形式で日付を自動入力するための処理です。

消去
N25 が結合セルになっているため、MergeArea を使って
結合されている範囲全体の内容をまとめて削除する処理です。

交通費
勤務区分が「有給」または「欠勤」の場合は交通費を加算せず、
それ以外の日だけ交通費（589円）を合計するための処理です。
最終的な交通費の合計は Q25 に出力します。
