# 給料計算ツール (Excel VBA)

このExcelファイルは、時給と勤務時間からその日の給料を自動計算するVBAツールです。

※ このツールは開発タブを操作する必要はありません。
   Excelを開いて「計算」「削除」「日付」ボタンを押すだけで使用できます。

## 使い方
1. 「給料計算.xlsm」ファイルを開く
2. 上部に表示されるセキュリティ警告の「コンテンツの有効化」を押す
3. 「このファイルを信頼済みドキュメントにしますか？」→「はい」を選択
4. カレンダーの日付の下のセルをクリックすると、右側にプルダウンが表示されます
5. 勤務内容を選択します
   - 「定時」：1日8時間勤務
   - 「残業◯」：残業時間に応じて自動計算
   - 「1時間〜7時間」：遅刻・早退に対応
   - 「有給」「欠勤」「休日出勤」にも対応
6. 選択が終わったら「計算」ボタンを押すと、その日の給料が自動計算されます

## その他の機能
- 「削除」ボタン：曜日・日付をすべて初期化します
- 「日付」ボタン：その月の日付を自動入力します  
   ※ K29セルに、その月の「曜日番号」を入力しておく必要があります  
     （例：日曜日=3、月曜日=4…）  
   ※ 曜日番号は、日付を正しく並べるために使用しています。  
     不要な曜日番号が表示された場合は、選択してDeleteしてください。
